## Version Naming Guide

- **Major (X.0.0)**: Breaking changes, major redesigns, significant new features
- **Minor (0.X.0)**: New features, new pages, new utilities (backwards compatible)
- **Patch (0.0.X)**: Bug fixes, minor improvements, content updates

## Categories

- **Added**: New features, pages, components
- **Changed**: Changes to existing functionality
- **Deprecated**: Features that will be removed in future versions
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security improvements
